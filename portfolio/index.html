  • p5.js Audio Visualizer

    Meant for desktop, this audio visualizer has three methods of demonstrating frequency and amplitude. There is a preloaded song list, or the user can drag and drop any audio file to play it.

    Allow for loading.

  • p5.js Color Picker

    Meant for desktop, this sketch loads a JSON file, and randomly places dots. Mousing over these dots shows the color names, as well as similar colors. Pressing up or down on a keyboard will lighten or darken the background.

  • p5.js Audio Visualizer (#2)

    This desktop only visualizer focuses on showing music in a way that mimics the vibrant album art of the preloaded songs. Because of keyboard commands, this can't be used on mobile at the moment.

    Allow for loading.

  • p5.js Sound Experiment

    Created for my Audio Foundations class, this project experimented with sound by loading multiple music files that could be played simultaniously, and a pitch slider.

    Allow for loading.
